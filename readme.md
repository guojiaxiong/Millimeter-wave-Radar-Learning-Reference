# 毫米波雷达学习资料汇总（Millimeter-wave Radar Learning Reference）

本仓库收录了所有与毫米波雷达相关的MATLAB函数、部分实验/仿真数据及测试源码，便于教研室成员和新同学快速学习和参考，也欢迎其他领域的开发者共同使用与交流。

This repository collects all MATLAB functions, selected experimental/simulation data, and test source codes related to millimeter-wave radar. It is designed to help lab members and newcomers quickly learn and reference these materials. Other developers are also welcome to use and contribute.

---

---

## 1. MATLAB函数 (MATLAB Functions)

本仓库包含各类常用的毫米波雷达信号处理、算法实现等MATLAB脚本、函数文件，适合快速调用或学习。

All commonly used MATLAB scripts and function files for millimeter-wave radar signal processing and algorithm implementations are included for quick access and learning.

---

## 2. 实验和仿真数据 (Experimental / Simulation Data)

部分典型实验结果和仿真数据也一并收录，便于复现实验和对比学习。

Some representative experimental and simulation data are provided for easier replication and comparative studies.

---

## 3. 测试源码 (Test Source Code)

相关的测试工程文件和调试代码也开放共享，可用来验证算法实现或辅助学习。

Associated test project files and debug codes are shared for validating algorithm implementations or for additional study support.

---

## 4. 资料持续更新 (Continuous Updates)

欢迎教研室成员和其他用户补充和更新更多资料。每次提交后请注明变更内容及用途。

All users are encouraged to update or add more resources. Please describe the purpose and content of each contribution in your commit.
